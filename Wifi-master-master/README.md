
## Wifi-master

该功能做TV开发时估计有用，功能和UI仅作参考，需要自己根据设计图进行改造

## 功能

1. Android wifi列表搜索、监听、连接(包括有密码和无密码连接)、wifi详情信息、添加隐藏网络
2. Android 热点开启关闭，适配Android 6.0,7.0,8.0
3. Android TV ListView条目焦点控制，任意控制条目的样式
